# MHServices - Site Vitrine Rénovation

## Problème Original
Site vitrine ultra moderne pour entreprise de rénovation bâtiment "MHServices" basée à Paris & Île-de-France. Tous types de rénovation intérieur/extérieur avec un design premium jaune + blanc.

## Architecture
- **Frontend**: React + Tailwind CSS + Framer Motion
- **Backend**: FastAPI (minimal, site vitrine)
- **Design System**: Barlow Condensed (titres) + Manrope (corps)
- **Animations**: Framer Motion pour scroll reveal et micro-interactions

## User Personas
- Particuliers en Île-de-France cherchant des services de rénovation
- Propriétaires souhaitant transformer leur intérieur/extérieur
